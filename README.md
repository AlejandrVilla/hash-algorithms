# Calculadora de algoritmos Hash
* MD4
* MD5
* SHA1
* SHA256
* CHMAC

## Dependencias
- Python 3.10
- GNU bash, version 5.1
- C++ 11

## Uso

1. para ayuda
    ```shell
    ./main.sh -h
    ```
2. para compilar los archivos y crear ejecutables
    ```shell
    ./main.sh -c
    ```
3. para ejecutar calculadora
    ```shell
    ./main.sh -e
    ```
